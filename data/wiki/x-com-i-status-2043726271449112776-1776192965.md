# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in the browser. The error message indicates that JavaScript is a mandatory requirement for using the platform, and provides guidance for users to enable it or switch to a supported browser.

## Overview

X.com requires [[JavaScript]] to function properly. When users attempt to access the platform with JavaScript disabled, they encounter an error page that prevents them from using the service. This is a common security and functionality requirement for modern web applications.

## Error Description

The error message displayed to users states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

This error occurs as a protective measure and indicates that the platform cannot load properly without JavaScript support.

## Resolution Steps

Users encountering this error have two primary options:

### Enable JavaScript

Users can enable [[JavaScript]] in their browser settings:
- Consult the browser's preferences or settings menu
- Locate the security or content settings section
- Enable JavaScript execution

### Switch Browsers

Users can alternatively switch to a [[web browser]] that supports JavaScript. X.com maintains a list of supported browsers in their Help Center.

## Related Technical Considerations

### Browser Compatibility

X.com provides a list of supported browsers in the [[Help Center]]. Users should ensure they are using a compatible browser version.

### Browser Extensions

Certain [[browser extensions]], particularly those related to privacy protection, may interfere with X.com functionality. The error page recommends disabling privacy-related extensions if issues persist after enabling JavaScript.
