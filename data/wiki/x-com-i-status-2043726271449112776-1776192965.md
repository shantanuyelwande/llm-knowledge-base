# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in a web browser. The error message indicates that JavaScript is a mandatory requirement for using the platform and provides guidance for resolution.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, they receive an error page that prevents further use of the service.

## Error Details

### Error Message

The primary error message states:

> We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com.

### Cause

The error occurs when:
- [[JavaScript]] is explicitly disabled in browser settings
- A user is accessing X.com through a browser that does not support JavaScript
- Browser extensions related to privacy may interfere with JavaScript functionality

### Technical Requirements

X.com requires users to:
1. Enable [[JavaScript]] in their browser settings
2. Use a [[Web Browser|supported web browser]]
3. Disable certain [[Privacy Extension|privacy-related extensions]] that may cause conflicts

## Resolution Steps

Users encountering this error can resolve it by:

1. **Enabling JavaScript**: Access browser settings and enable JavaScript functionality
2. **Switching Browsers**: Use a browser listed in X.com's supported browsers list
3. **Disabling Extensions**: Remove or disable privacy-related extensions that may interfere with X.com functionality
4. **Consulting Help Resources**: Reference the [X.com Help Center](https://help.x.com/using-x/x-supported-browsers) for detailed instructions
