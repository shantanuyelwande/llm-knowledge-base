# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) without JavaScript enabled in the browser. The error page prevents users from accessing the platform and provides guidance on enabling JavaScript or switching to a supported browser.

## Overview

When attempting to access X.com with [[JavaScript]] disabled, users encounter an error message indicating that the platform requires JavaScript to function. This is a common [[web browser]] compatibility issue that affects the accessibility of the platform across different browsing configurations.

## The Error Message

The primary error notification states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

This message appears when the platform detects that [[JavaScript]] is not active in the user's [[web browser]], preventing the site from loading its interactive features and functionality.

## Troubleshooting Solutions

### Enable JavaScript

Users experiencing this error are advised to enable [[JavaScript]] in their browser settings. The specific steps vary depending on the browser being used.

### Switch Browsers

X.com provides a list of [[supported browsers]] in their Help Center. Users encountering persistent issues with their current browser may need to switch to an officially supported alternative.

### Disable Privacy Extensions

The error page notes that some [[privacy-related browser extensions]] may interfere with X.com's functionality. Users are advised to:

- Identify and disable privacy-related extensions
- Refresh the page after disabling extensions
- Re-enable extensions selectively if needed
