# X.com JavaScript Requirement

## Summary

This article documents the JavaScript requirement for accessing X.com (formerly Twitter) and the error message users encounter when JavaScript is disabled in their browser.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, they receive an error message informing them that JavaScript is necessary to continue using the service.

## JavaScript Requirement

### Why JavaScript is Required

X.com relies on [[JavaScript]] as a core technology for rendering content, handling user interactions, and managing dynamic functionality on the platform. Without JavaScript enabled, the service cannot load properly or provide its full feature set.

### Error Message

When JavaScript is disabled, users see the following notification:

> We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com.

## Troubleshooting

### Enabling JavaScript

Users experiencing this issue have two primary options:

1. **Enable JavaScript** - Users can enable JavaScript in their browser settings
2. **Switch Browsers** - Users can switch to a [[Web browser|supported browser]] that has JavaScript enabled

### Browser Support

X.com maintains a list of [[Web browser|supported browsers]] in their Help Center. Users should consult this resource to ensure their browser meets the platform's requirements.

### Privacy Extensions

Some [[Browser extension|privacy-related browser extensions]] may interfere with X.com's functionality. The platform recommends disabling these extensions if users encounter issues accessing the service.
