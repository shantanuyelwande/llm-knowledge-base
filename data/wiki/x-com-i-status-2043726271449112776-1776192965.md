# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in the browser. The error page indicates that JavaScript is a mandatory requirement for using the platform and provides guidance for resolution.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents further browsing and interaction with the service.

## Error Details

### Error Message

The primary error message states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Cause

The error occurs when:
- [[JavaScript]] is disabled in the browser settings
- The browser does not support JavaScript functionality
- Browser extensions related to privacy or security block JavaScript execution

## Resolution Steps

Users encountering this error have several options:

1. **Enable JavaScript** - Activate JavaScript in the browser's settings or preferences
2. **Switch Browsers** - Use a browser that supports JavaScript from the list of [[X.com Supported Browsers]]
3. **Disable Extensions** - Remove privacy-related browser extensions that may interfere with JavaScript execution

### Browser Compatibility

X.com provides a list of officially supported browsers in their [[Help Center]]. Users should verify their browser is on this list before troubleshooting other issues.
