# X.com JavaScript Requirement Error

## Summary

This article documents a browser error encountered when accessing X.com (formerly Twitter) without JavaScript enabled. The error page explains that JavaScript is required to use the platform and provides guidance for resolution.

## Overview

X.com requires [[JavaScript]] to be enabled in the user's browser to function properly. When JavaScript is disabled, users attempting to access the platform receive an error message instead of the normal interface.

## Error Details

### Error Message

The primary error message states:

> We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com.

### Common Causes

- JavaScript is manually disabled in browser settings
- Browser extensions, particularly [[privacy-related extensions]], may interfere with JavaScript functionality
- Outdated or unsupported browser version

## Resolution Steps

### Enable JavaScript

Users can resolve this issue by:

1. Enabling JavaScript in their browser's security settings
2. Disabling privacy-related extensions that may block JavaScript execution
3. Switching to a [[supported browser]] (see X.com Help Center for current list)

### Browser Compatibility

X.com maintains a list of supported browsers in their Help Center. Users experiencing issues should verify their browser is on the supported list.
