# X.com JavaScript Requirement

## Summary
This article documents the JavaScript requirement error encountered when accessing X.com (formerly Twitter) with JavaScript disabled. The platform requires JavaScript to function and provides guidance for users experiencing this technical issue.

## Overview

X.com is a social media platform that requires [[JavaScript]] to be enabled in the user's web browser to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error message preventing them from using the service.

## Technical Requirements

### JavaScript Dependency
X.com requires [[JavaScript]] to be active in the browser. This is a fundamental technical requirement for the platform's functionality. Users who have JavaScript disabled will be unable to access any features of the site.

### Supported Browsers
The platform maintains a list of supported browsers in its Help Center. Users experiencing issues should verify they are using a supported browser before troubleshooting other potential problems.

## Troubleshooting

### Common Issues
When JavaScript is disabled, users will see an error message indicating that JavaScript must be enabled to continue using X.com. The platform provides several troubleshooting steps:

1. **Enable JavaScript** - Users should enable JavaScript in their browser settings
2. **Switch Browsers** - If the current browser is not supported, users should switch to a supported browser
3. **Disable Privacy Extensions** - Some [[privacy]] related browser extensions may interfere with X.com functionality and should be temporarily disabled
4. **Try Again** - After making changes, users should refresh the page and attempt to access the site again
