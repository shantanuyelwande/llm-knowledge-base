# X.com JavaScript Requirement

## Summary
This article documents an error message encountered when accessing X.com (formerly Twitter) without [[JavaScript]] enabled in the browser. The platform requires JavaScript functionality to operate and displays a specific error page when the requirement is not met.

## Overview

X.com is a social media platform that requires [[JavaScript]] to be enabled in users' web browsers in order to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error message rather than the normal interface.

## Error Message Details

### Content
The error notification informs users that:
- JavaScript has been detected as disabled in their browser
- JavaScript must be enabled to continue using X.com
- Users should either enable JavaScript or switch to a supported browser
- A list of supported browsers is available in the [[Help Center]]

### Technical Context
- **URL:** https://x.com/i/status/2043726271449112776
- **Detection Method:** Client-side JavaScript detection
- **Error Type:** Browser compatibility/configuration issue

## Troubleshooting

### User Actions
Users encountering this error can:

1. **Enable JavaScript** - Modify browser settings to allow JavaScript execution
2. **Switch Browsers** - Use a [[web browser|supported browser]] that is compatible with X.com
3. **Consult Help Resources** - Reference the X.com Help Center for detailed browser compatibility information
4. **Disable Extensions** - Remove privacy-related browser extensions that may interfere with X.com functionality

### Known Issues
Some [[privacy]] related extensions may cause JavaScript-related issues on X.com and should be disabled before attempting to access the platform again.
