# X.com JavaScript Requirement

## Summary

This article documents the JavaScript requirement for accessing X.com (formerly Twitter), including the error message users encounter when JavaScript is disabled and the troubleshooting steps available.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. Users who have disabled JavaScript in their browser will encounter an error message preventing them from accessing the platform's features.

## Error Message and Requirements

When JavaScript is disabled, X.com displays the following message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

This indicates that [[JavaScript]] is a mandatory technical requirement for the platform's operation.

## Supported Browsers

Users experiencing JavaScript-related issues are directed to consult the Help Center for a complete list of [[supported browsers]]. X.com maintains compatibility requirements with specific browser versions and configurations.

## Troubleshooting

### Enabling JavaScript

Users should enable JavaScript in their browser settings to restore access to X.com.

### Browser Compatibility

If JavaScript cannot be enabled, users are advised to switch to a [[supported browser]]. The platform provides documentation on compatible browsers through its Help Center.

### Privacy Extensions

Some privacy-related browser extensions may cause compatibility issues with X.com. The platform recommends disabling such extensions and attempting to access the service again.
