# X.com JavaScript Requirement Error

## Summary

This article documents a technical error message encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in a web browser. The error prevents users from accessing the platform and requires enabling JavaScript or switching to a supported browser to continue.

## Overview

X.com displays a JavaScript availability error when users attempt to access the platform without JavaScript enabled in their browser. This is a technical requirement of the modern X.com platform, which relies on client-side [[JavaScript]] for core functionality.

## Error Details

### Error Message

The primary error notification states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Cause

The error occurs when:
- [[JavaScript]] is disabled in the user's browser settings
- The browser does not support required JavaScript features
- Browser extensions related to privacy or security block JavaScript execution

### Privacy Extensions Notice

X.com notes that certain privacy-related browser extensions may interfere with platform functionality. The system recommends:
- Disabling privacy extensions temporarily
- Adding X.com to extension whitelists
- Attempting to access the site again after disabling extensions

## Resolution Steps

Users encountering this error can resolve it through:

1. **Enable JavaScript** - Activate JavaScript in browser settings
2. **Switch Browsers** - Use a [[Web Browser|supported browser]] from X.com's approved list
3. **Disable Extensions** - Temporarily disable privacy or security extensions
4. **Consult Help Resources** - Visit the X.com Help Center for browser compatibility information
