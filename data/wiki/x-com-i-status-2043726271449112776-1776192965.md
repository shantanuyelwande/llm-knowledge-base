# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) without JavaScript enabled in the browser. The error page provides guidance for users who need to enable JavaScript or switch to a supported browser to use the platform.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, the site displays an error message preventing further navigation and usage of the service.

## Error Message

When JavaScript is not available, users encounter the following notification:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

## Resolution Steps

Users experiencing this issue can resolve it through the following methods:

### Enable JavaScript

Users should enable JavaScript in their browser settings. The process varies depending on the browser being used.

### Switch Browsers

X.com recommends switching to a [[Supported Browser|supported browser]] if JavaScript cannot be enabled. A complete list of compatible browsers is available in the X.com Help Center.

### Disable Privacy Extensions

Some [[Browser Extension|privacy-related browser extensions]] may interfere with X.com functionality. Disabling these extensions can resolve the issue:

- Privacy-focused extensions may block JavaScript execution
- Users should temporarily disable extensions and retry access
- Re-enable extensions after successfully accessing X.com if desired

## Technical Requirements

X.com has specific technical requirements for proper operation:

- **JavaScript**: Required for core functionality
- **Supported Browsers**: Limited to browsers meeting X.com's compatibility standards
- **Browser Extensions**: Certain privacy extensions may cause conflicts
