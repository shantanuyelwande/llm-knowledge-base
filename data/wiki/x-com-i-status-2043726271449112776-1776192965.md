# X.com JavaScript Requirement

## Summary

This article documents the technical requirement that [[JavaScript]] must be enabled to access X.com (formerly Twitter). When JavaScript is disabled in a web browser, users encounter an error message preventing access to the platform.

## Overview

X.com requires [[JavaScript]] to function properly. Users attempting to access the platform with JavaScript disabled will encounter a blocking error message that prevents further navigation.

## Technical Requirements

### JavaScript Dependency

X.com is built as a [[JavaScript]]-dependent web application. This means:

- JavaScript must be enabled in the user's browser
- The platform cannot be accessed through browsers with JavaScript disabled
- Disabling JavaScript prevents the application from loading and functioning

### Supported Browsers

Users experiencing access issues should verify they are using a [[Web Browser|supported browser]]. A complete list of compatible browsers is available in the X.com Help Center.

## Error Messages and Troubleshooting

### Primary Error Message

When JavaScript is disabled, users see:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Browser Extensions

Certain [[Browser Extension|privacy-related browser extensions]] may cause compatibility issues with X.com, even when JavaScript is enabled. The platform recommends:

- Disabling privacy extensions temporarily
- Testing access after disabling extensions
- Re-enabling extensions selectively if the issue is resolved

### Recovery Options

Users encountering errors should:

1. Enable JavaScript in their browser settings
2. Switch to a supported browser
3. Disable problematic browser extensions
4. Consult the Help Center for additional support
