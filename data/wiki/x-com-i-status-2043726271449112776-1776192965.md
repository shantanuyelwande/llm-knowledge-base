# X.com JavaScript Requirement Error

## Summary
This article documents a browser error encountered when accessing X.com (formerly Twitter) without JavaScript enabled. The platform requires JavaScript functionality to operate and displays an error message when users attempt to access it with JavaScript disabled.

## Overview

X.com is a [[social media]] platform that requires [[JavaScript]] to function properly. When users attempt to access the platform with JavaScript disabled in their [[web browser]], they encounter an error page that prevents them from using the service.

## Error Message and Resolution

### The Error
Users who have disabled JavaScript or are using an unsupported browser will see the following message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Recommended Solutions

1. **Enable JavaScript**: Users can enable JavaScript in their browser settings
2. **Switch Browsers**: Users can switch to a [[supported browser]] listed in the X.com Help Center
3. **Disable Privacy Extensions**: Some [[privacy-related browser extensions]] may interfere with X.com functionality and should be disabled

## Technical Requirements

X.com requires the following for proper operation:

- **JavaScript**: Must be enabled in the browser
- **Supported Browser**: Users must use a browser officially supported by X.com
- **Browser Extensions**: Certain privacy extensions may cause compatibility issues
