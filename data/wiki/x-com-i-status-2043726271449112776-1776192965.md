# X.com JavaScript Requirement

## Summary

This article documents the [[JavaScript]] requirement for accessing X.com (formerly Twitter) and the error message displayed when JavaScript is disabled in a web browser.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access X.com with JavaScript disabled, the platform displays an error message preventing further access to the service.

## Technical Requirements

### JavaScript Dependency

X.com is built as a [[Single Page Application|SPA]] that relies on [[JavaScript]] for core functionality. The platform cannot render content or provide interactive features without JavaScript enabled in the browser.

### Browser Support

X.com maintains a list of officially supported browsers that meet the JavaScript and performance requirements needed for the platform. Users experiencing issues are directed to consult the Help Center for a complete list of compatible browsers.

## Error Handling

### JavaScript Disabled Error

When JavaScript is not available, users encounter a blocking error message that states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

The error page provides users with two primary options:
- Enable JavaScript in their browser settings
- Switch to a supported browser

### Browser Extensions

The error page notes that certain [[privacy-focused browser extensions]] may interfere with X.com functionality, even when JavaScript is enabled. Users are advised to disable privacy-related extensions and retry accessing the platform.

## User Solutions

Users encountering this error can:

1. **Enable JavaScript** - Access browser settings and enable JavaScript execution
2. **Switch Browsers** - Use an officially supported browser from X.com's approved list
3. **Disable Extensions** - Remove or disable privacy-related browser extensions that may conflict with the platform
4. **Contact Support** - Access the [[Help Center]] for additional troubleshooting
