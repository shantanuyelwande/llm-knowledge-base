# X.com JavaScript Requirement Error

## Summary
This article documents a browser error message encountered when attempting to access X.com (formerly Twitter) without JavaScript enabled. The error indicates that JavaScript is a mandatory requirement for using the X.com platform and provides guidance for users to enable it or switch to a supported browser.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents further navigation and usage of the service.

## Error Message Details

### Primary Error
The error message states: "JavaScript is not available. We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

This indicates that:
- [[JavaScript]] execution is mandatory for X.com functionality
- The browser has detected that JavaScript support is either disabled or unavailable
- The user must take corrective action to proceed

## Resolution Steps

Users encountering this error can resolve it through the following methods:

1. **Enable JavaScript** - Activate JavaScript support in the browser settings
2. **Switch Browsers** - Use an alternative browser with full JavaScript support
3. **Consult Support** - Review the [[Help Center]] for a list of [[supported browsers]]

## Technical Considerations

### Browser Compatibility
X.com maintains a list of officially supported browsers in its Help Center. Users experiencing issues should verify their browser is on this approved list.

### Privacy Extensions
The error page notes that "some privacy related extensions may cause issues on x.com." Users should disable such extensions and attempt to reload the page if problems persist.
