# X.com JavaScript Requirement Error

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) without [[JavaScript]] enabled in the browser. The error message indicates that JavaScript is a mandatory requirement for using the platform's web interface.

## Overview
X.com requires [[JavaScript]] to function properly. When users attempt to access the platform with JavaScript disabled, they encounter an error page that prevents further browsing and interaction with the service.

## Error Details

### Error Message
The standard error notification states:
> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Root Cause
[[JavaScript]] is a core technology that X.com relies upon for:
- Dynamic content rendering
- User interface interactivity
- Real-time updates and notifications
- Session management

## Resolution Steps

Users encountering this error can resolve it through several methods:

1. **Enable JavaScript** - Modify browser settings to allow JavaScript execution
2. **Switch Browsers** - Use a [[web browser]] that supports JavaScript and is on X.com's supported list
3. **Check Browser Extensions** - Disable privacy-related extensions that may block JavaScript

### Privacy Extension Issues
The error page specifically notes that some [[privacy extension|privacy-related browser extensions]] may interfere with X.com functionality and should be temporarily disabled.

## Technical Requirements

### Supported Browsers
X.com maintains a list of [[supported web browsers]] in their Help Center. Users should verify their browser version meets current requirements.

### Browser Capabilities
Supported browsers must include:
- [[JavaScript]] engine support
- [[Cookie]] handling capability
- [[DOM]] manipulation support
- [[AJAX]] functionality
