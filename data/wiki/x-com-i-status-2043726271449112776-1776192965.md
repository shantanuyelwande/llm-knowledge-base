# X.com JavaScript Requirement Error

## Summary
This article documents a browser error encountered when accessing X.com (formerly Twitter) without JavaScript enabled. The platform requires JavaScript to function and displays an error message when users attempt to access it with JavaScript disabled.

## Overview

X.com is a [[social media platform]] that requires [[JavaScript]] to be enabled in users' web browsers to function properly. When JavaScript is disabled, users encounter an error page that prevents access to the platform's features and content.

## Error Message and Details

### Error Description
When JavaScript is disabled, X.com displays the following error message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Technical Context
- **Detection Method:** X.com automatically detects when JavaScript is not available
- **User Impact:** Complete access denial to the platform
- **Error Type:** Browser compatibility issue

## Resolution Steps

Users experiencing this error can resolve it through the following methods:

### 1. Enable JavaScript
Users should enable JavaScript in their browser settings. The specific steps vary depending on the browser being used.

### 2. Switch Browsers
If JavaScript cannot be enabled, users should switch to a [[supported browser]]. X.com maintains a list of compatible browsers in their Help Center.

### 3. Disable Privacy Extensions
Some [[privacy-related browser extensions]] may interfere with X.com's JavaScript functionality. The platform recommends disabling these extensions before attempting to access the service again.

## Browser Compatibility

X.com provides documentation on supported browsers through their Help Center. Users experiencing compatibility issues should consult this resource for a complete list of [[supported browsers]] and system requirements.
