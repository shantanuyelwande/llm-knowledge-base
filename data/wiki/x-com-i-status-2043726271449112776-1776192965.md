# X.com JavaScript Requirement Error

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) with JavaScript disabled in the browser. The platform requires JavaScript to function and displays an error message prompting users to enable it or switch to a supported browser.

## Overview

X.com is a web-based platform that requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents them from accessing X.com's services.

## Error Message and Content

### Primary Error
The platform displays a notification stating: "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Recommended Actions
Users encountering this error are advised to:
1. Enable [[JavaScript]] in their browser settings
2. Switch to a [[Web Browser|supported web browser]]
3. Consult the Help Center for a list of compatible browsers

## Browser Compatibility

X.com maintains a list of [[Web Browser|supported browsers]] that are compatible with the platform's technical requirements. Users experiencing issues are directed to the Help Center for current compatibility information.

## Privacy Extensions Notice

The error page includes a warning that "Some privacy related extensions may cause issues on x.com." Users are advised to disable privacy-related browser extensions and attempt to access the platform again if they continue to experience difficulties.
