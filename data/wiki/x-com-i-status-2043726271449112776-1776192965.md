# X.com JavaScript Requirement Error

## Summary
This article documents a technical error message encountered when accessing X.com (formerly Twitter) with JavaScript disabled in the browser. The error prevents users from accessing the platform and directs them to enable JavaScript or switch to a supported browser.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that explains the requirement and provides guidance for resolution.

## Error Message Details

### Primary Error
The main error message states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

This error indicates that the X.com platform has a hard dependency on JavaScript for core functionality and cannot operate without it enabled.

### Secondary Warning
An additional notice warns users:

> "Some privacy related extensions may cause issues on x.com. Please disable them and try again."

This suggests that certain [[privacy]] and [[security]] browser extensions may interfere with JavaScript execution or X.com's functionality, even when JavaScript is technically enabled.

## Resolution Steps

Users encountering this error have two primary options:

1. **Enable JavaScript** - Modify browser settings to allow JavaScript execution
2. **Switch Browsers** - Use a supported browser from X.com's list of compatible options

### Supported Browsers

X.com maintains a list of officially supported browsers. Users can consult the Help Center for current compatibility information.
