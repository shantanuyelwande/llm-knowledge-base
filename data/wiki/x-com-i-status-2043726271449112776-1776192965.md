# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in the browser. The error page provides guidance for users experiencing access issues due to missing JavaScript support.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, the browser displays an error message preventing further use of the service.

## Error Message

The primary error notification states:

> We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com.

## Resolution Steps

Users encountering this issue can resolve it through the following methods:

### Enable JavaScript

The simplest solution is to enable [[JavaScript]] in the browser settings. Instructions for doing this vary depending on the specific browser being used.

### Switch Browsers

Users can alternatively switch to a [[web browser]] that supports JavaScript. X.com maintains a list of [[supported browsers]] in their Help Center for reference.

### Disable Privacy Extensions

The error page notes that some [[privacy-related browser extensions]] may interfere with X.com functionality:

> Some privacy related extensions may cause issues on x.com. Please disable them and try again.

Users experiencing persistent issues should consider temporarily disabling privacy extensions and attempting to access the site again.
