# X.com JavaScript Requirement

## Summary

This article documents the JavaScript requirement for accessing X.com (formerly Twitter), including the technical requirements, supported browsers, and troubleshooting information for users who encounter JavaScript disabled errors.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. Users who attempt to access the platform with JavaScript disabled will encounter an error message directing them to enable JavaScript or switch to a supported browser.

## Technical Requirements

### JavaScript Dependency

X.com is built as a modern web application that relies on [[JavaScript]] for core functionality. This is a common requirement for contemporary web platforms that require dynamic, interactive user interfaces.

### Supported Browsers

X.com maintains a list of officially supported browsers. Users experiencing issues are directed to consult the Help Center for the complete list of compatible browsers and their minimum version requirements.

## Error Messages and Troubleshooting

### Common Error Scenario

When JavaScript is disabled, users see the following error message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Enabling JavaScript

Users encountering this error should:
1. Enable JavaScript in their browser settings
2. Verify they are using a [[supported browser]]
3. Consult the X.com Help Center for detailed instructions

### Privacy Extensions

Some [[privacy-related browser extensions]] may interfere with X.com functionality. The platform recommends temporarily disabling privacy extensions and attempting to access the service again.
