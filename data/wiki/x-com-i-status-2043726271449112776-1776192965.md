# X.com JavaScript Requirement

## Summary
This article documents the JavaScript requirement error encountered on X.com, a social media platform. The error occurs when JavaScript is disabled in a user's browser, preventing access to the platform's functionality.

## Overview

X.com (formerly known as [[Twitter]]) requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error message that prevents them from using the service.

## Error Description

### JavaScript Disabled Error
When JavaScript is not available in the user's browser, X.com displays an error page with the following characteristics:

- **Primary Message**: "JavaScript is not available"
- **Explanation**: The platform has detected that JavaScript is disabled and cannot continue operation
- **Impact**: Users are unable to access X.com's features and content

### Cause
The error occurs due to:
- JavaScript being explicitly disabled in browser settings
- Use of an unsupported or outdated browser
- Browser extensions that block JavaScript execution

## Resolution

### User Actions
To resolve this issue, users should:

1. **Enable JavaScript**: Activate JavaScript in their browser's settings
2. **Switch Browsers**: Use a [[supported browser]] from X.com's approved list
3. **Disable Extensions**: Remove or disable privacy-related browser extensions that may interfere with JavaScript execution

### Supported Browsers
X.com maintains a list of supported browsers in their [[Help Center]]. Users experiencing issues should consult this resource for compatibility information.

## Related Information

### Platform Requirements
- **Technology**: [[JavaScript]] (required)
- **Browser Support**: Limited to approved, modern browsers
- **Extensions**: Some privacy extensions may cause compatibility issues
