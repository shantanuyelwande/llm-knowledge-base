# X.com JavaScript Requirement Error

## Summary
This article documents a technical error message encountered when accessing X.com (formerly Twitter) with JavaScript disabled in the user's browser. The error prevents access to the platform's core functionality and provides guidance for resolution.

## Overview

X.com requires [[JavaScript]] to be enabled in order to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that explains the issue and provides troubleshooting steps.

## Error Description

### Technical Issue
The error occurs when:
- [[JavaScript]] is disabled in the browser settings
- The user is accessing X.com through an unsupported or outdated browser
- Browser extensions related to privacy or security are blocking JavaScript execution

### Error Message
The platform displays a notification stating: "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

## Resolution Steps

### Enable JavaScript
Users experiencing this error should:
1. Enable JavaScript in their browser settings
2. Verify they are using a [[Web Browser|supported browser]]
3. Disable any privacy-related browser extensions that may interfere with JavaScript execution
4. Clear browser cache and cookies if the issue persists
5. Attempt to access X.com again

### Browser Compatibility
X.com maintains a list of supported browsers in their Help Center. Users with unsupported or outdated browsers should upgrade to a compatible version.
