# X.com JavaScript Requirement Error

## Summary
This article documents a technical error message encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in a web browser. The platform requires JavaScript to function properly and will display an error message directing users to enable it or switch to a supported browser.

## Overview

X.com is a social media platform that relies on [[JavaScript]] for core functionality. When users attempt to access the platform with JavaScript disabled, they encounter a blocking error that prevents further navigation.

## Error Message Details

### Primary Error
The main error notification states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

Users are directed to consult the Help Center for a list of [[supported browsers]].

### Secondary Notice
An additional privacy-related warning appears:

> "Some privacy related extensions may cause issues on x.com. Please disable them and try again."

This indicates that certain [[browser extensions]] designed for privacy protection may interfere with the platform's JavaScript execution.

## Technical Requirements

X.com requires:
- **JavaScript**: Enabled in browser settings
- **Supported Browser**: A modern browser compatible with X.com's technical specifications
- **Extensions**: Privacy extensions may need to be disabled or configured to allow X.com functionality

## Resolution Steps

Users encountering this error should:

1. Enable [[JavaScript]] in their browser settings
2. Verify their browser is on the [[list of supported browsers]]
3. Disable or whitelist privacy-related [[browser extensions]]
4. Clear browser cache and cookies if issues persist
5. Try accessing the platform again
