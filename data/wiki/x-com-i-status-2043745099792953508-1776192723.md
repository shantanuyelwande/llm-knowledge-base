# X.com JavaScript Requirement

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) without [[JavaScript]] enabled in the browser. The platform requires JavaScript to function and will display an error message directing users to enable it or switch to a supported browser.

## Overview

X.com is a [[social media]] platform that requires [[JavaScript]] to be enabled in web browsers for proper functionality. When a user attempts to access the site with JavaScript disabled, they encounter a technical barrier that prevents normal use of the platform.

## Technical Requirements

### JavaScript Dependency

X.com relies on [[JavaScript]] as a core technology for its web interface. This dependency means that:

- JavaScript must be enabled in the user's browser settings
- Users cannot access X.com's full functionality without it
- The platform will display an error message if JavaScript is disabled

### Supported Browsers

Users experiencing JavaScript-related issues are directed to consult X.com's Help Center for a list of officially supported browsers. The platform recommends using modern browsers with full JavaScript support.

## Error Message and Resolution

When JavaScript is disabled, users see the following message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Recommended Actions

1. **Enable JavaScript** - Users can enable JavaScript in their browser's security settings
2. **Switch Browsers** - Use an alternative browser from X.com's supported browsers list
3. **Disable Extensions** - Some privacy-related browser extensions may interfere with X.com functionality and should be disabled
4. **Retry Access** - After making changes, users should attempt to access the site again

## Related Issues

[[Browser extensions]], particularly those focused on privacy protection, may cause compatibility issues with X.com. Users experiencing persistent problems are advised to temporarily disable such extensions.
