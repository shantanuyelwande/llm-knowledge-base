# X.com JavaScript Requirement

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) without JavaScript enabled in the browser. The platform requires JavaScript to function properly and displays an error message directing users to enable it or switch to a supported browser.

## Overview

X.com, the social media platform operated by [[X Corp]], requires [[JavaScript]] to be enabled in users' web browsers to access its services. When users attempt to access the platform with JavaScript disabled, they encounter a blocking error page rather than the standard X.com interface.

## Technical Requirements

### JavaScript Dependency
The X.com platform is built with [[JavaScript]] as a core requirement for its functionality. This is common among modern [[Single Page Application|single-page applications]] that rely on client-side rendering and dynamic content loading.

### Browser Compatibility
X.com maintains a list of supported browsers in its [[Help Center]]. Users experiencing issues are directed to:
- Enable JavaScript in their current browser settings, or
- Switch to an officially supported browser

## Error Message and User Experience

When JavaScript is disabled, users see:
- A notification that JavaScript is not available
- Instructions to enable JavaScript or switch browsers
- A link to the Help Center for supported browser information
- An option to retry the connection

### Privacy Extensions
The error messaging notes that some [[Privacy Extension|privacy-related browser extensions]] may interfere with X.com's functionality. Users encountering persistent issues are advised to temporarily disable such extensions before retrying.
