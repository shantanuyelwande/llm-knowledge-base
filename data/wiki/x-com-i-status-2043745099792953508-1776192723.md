# X.com JavaScript Requirement Error

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) without [[JavaScript]] enabled in the browser. The error message indicates that JavaScript is a mandatory requirement for using the X.com platform.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they receive an error message preventing them from continuing. This is a common technical requirement for modern web applications that rely on client-side scripting.

## Error Details

### Error Message
The platform displays a notification stating: "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Causes
- [[JavaScript]] is disabled in the browser settings
- Browser does not support JavaScript execution
- Browser extensions or privacy tools may block JavaScript functionality

### Troubleshooting Steps

Users encountering this error can take the following actions:

1. **Enable JavaScript**: Activate JavaScript in browser settings
2. **Switch Browsers**: Use a [[Supported Web Browsers|supported browser]] from X.com's compatibility list
3. **Disable Extensions**: Remove or disable [[Browser Extensions|privacy-related browser extensions]] that may interfere with JavaScript execution
4. **Clear Cache**: Try clearing browser cache and cookies before attempting again

## Browser Compatibility

X.com maintains a list of [[Supported Web Browsers|supported browsers]] in their Help Center. Users experiencing issues should verify their browser is on the approved list.
