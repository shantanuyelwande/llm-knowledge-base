# X.com JavaScript Requirement Error

## Summary

This article documents a browser error message encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled. The error prevents users from accessing the platform and provides guidance on enabling JavaScript or switching to a supported browser.

## Overview

When attempting to access X.com without [[JavaScript]] enabled in the browser, users encounter a blocking error page that prevents further navigation of the platform. This is a technical requirement of the modern X.com web application.

## Error Details

### JavaScript Disabled Detection

X.com's platform requires [[JavaScript]] to function properly. When the browser has JavaScript disabled, the site displays an error message stating:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Error Resolution

Users encountering this error have two primary options:

1. **Enable JavaScript** - Re-enable JavaScript in the browser's settings
2. **Switch Browsers** - Use a [[Web Browser|supported browser]] compatible with X.com

### Browser Support

X.com maintains a list of supported browsers in their [[Help Center]]. Users experiencing compatibility issues should consult this resource for current information on which browsers meet the platform's requirements.

## Related Technical Issues

### Privacy Extensions

Some [[Privacy Extension|privacy-related browser extensions]] may cause issues when accessing X.com. The error message recommends disabling these extensions as a troubleshooting step if users continue experiencing problems after enabling JavaScript.

### Troubleshooting

The platform provides a "Try again" option for users to retry loading the page after making necessary changes to their browser configuration.
