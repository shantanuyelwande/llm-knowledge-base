# JavaScript Requirement Error on X.com

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) with JavaScript disabled in the web browser. The platform requires JavaScript to function and displays an error message directing users to enable it or switch to a supported browser.

## Overview

X.com is a web-based social media platform that requires [[JavaScript]] to be enabled in order to function properly. When users attempt to access the platform with JavaScript disabled, they encounter a blocking error message that prevents further interaction with the site.

## Error Details

### Error Message
The primary error message displayed states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Cause
This error occurs when:
- [[JavaScript]] is disabled in the user's browser settings
- The browser does not support JavaScript execution
- Browser extensions related to privacy or security are blocking JavaScript execution

### Error Resolution

Users encountering this error have several options:

1. **Enable JavaScript** - Modify browser settings to allow JavaScript execution
2. **Switch Browsers** - Use a [[web browser]] that supports JavaScript and is officially supported by X.com
3. **Disable Extensions** - Remove or disable privacy-related browser extensions that may interfere with JavaScript functionality

## Supported Browsers

X.com provides a list of supported browsers in their Help Center. Users experiencing issues are directed to consult this resource for compatibility information.

[[Web browsers]] require [[JavaScript]] to execute dynamic code and provide interactive features on modern websites. X.com relies on JavaScript for core functionality including:
- User interface rendering

- Content loading and display
