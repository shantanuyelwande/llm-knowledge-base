# X.com JavaScript Requirement Error

## Summary

This article documents a common technical error encountered when accessing X.com (formerly Twitter) with JavaScript disabled in a web browser. The platform requires JavaScript to function properly and displays an error message directing users to enable it or switch to a supported browser.

## Overview

X.com is a web application that depends on [[JavaScript]] to render its user interface and provide interactive functionality. When a user attempts to access the platform with JavaScript disabled, they encounter an error page rather than the normal X.com experience.

## Error Message and Resolution

### The Error

Users who have disabled [[JavaScript]] in their browser receive the following message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Recommended Solutions

The platform offers two primary solutions:

1. **Enable JavaScript** - Users can re-enable JavaScript in their browser settings
2. **Switch Browsers** - Users can switch to a [[Web Browser|supported browser]] that has JavaScript enabled

A complete list of supported browsers is available in the X.com [[Help Center]].

## Browser Compatibility

X.com maintains a list of officially supported browsers in its Help Center documentation. Users experiencing issues should verify their browser is on the approved list and that JavaScript is properly enabled.

## Additional Technical Issues

### Privacy Extensions

The error page notes that some [[Privacy Extension|privacy-related browser extensions]] may cause functionality issues on X.com, even when JavaScript is enabled. Users experiencing problems are advised to:

- Disable privacy-related extensions
- Clear browser cache
- Attempt to access the platform again
