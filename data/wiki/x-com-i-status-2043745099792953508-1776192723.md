# X.com JavaScript Requirement Error

## Summary
This article documents a technical error encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in a web browser. The platform requires JavaScript to function properly and displays an error message prompting users to enable it or switch to a supported browser.

## Overview

X.com is a [[social media]] platform that requires [[JavaScript]] to be enabled in order to access its features and content. When a user attempts to visit the platform with JavaScript disabled, they encounter an error page that prevents them from proceeding.

## Technical Requirements

### JavaScript Dependency
X.com depends on [[JavaScript]] as a core technology for its functionality. The platform cannot load or display content without JavaScript enabled in the user's [[web browser]].

### Supported Browsers
Users experiencing this error are directed to consult the Help Center for a list of officially supported browsers. The platform recommends using modern browsers that have full JavaScript support.

## Error Resolution

### User Actions
When encountering this error message, users have two primary options:

1. **Enable JavaScript** - Users can enable JavaScript in their browser settings to continue using X.com
2. **Switch Browsers** - Users can switch to a [[web browser]] that is officially supported by X.com

### Privacy Extensions
The error message notes that some privacy-related browser extensions may interfere with X.com's functionality. Users experiencing issues are advised to:
- Disable privacy extensions temporarily
- Attempt to reload the page
- Re-enable extensions after confirming access works
