# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) with [[JavaScript]] disabled in the browser. The error page provides guidance for users to enable JavaScript or switch to a supported browser to continue using the platform.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents further navigation until JavaScript is enabled or a compatible browser is used.

## Error Details

### JavaScript Disabled Detection

The platform automatically detects when [[JavaScript]] is disabled and displays a notification message:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Error Resolution

Users encountering this error have two primary options:

1. **Enable JavaScript** - Activate JavaScript in the browser's settings
2. **Switch Browsers** - Use a [[Supported Browsers|supported browser]] that has JavaScript enabled by default

## Troubleshooting

### Browser Compatibility

X.com maintains a list of supported browsers in their Help Center. Users experiencing issues are directed to check this resource for compatibility information.

### Privacy Extensions

The error page notes that some [[Privacy Extensions|privacy-related extensions]] may interfere with X.com functionality. The recommended troubleshooting step is to:

- Disable privacy-related browser extensions
- Attempt to access X.com again

### General Troubleshooting

If users encounter persistent issues, the platform provides a "Try again" button to retry the connection.
