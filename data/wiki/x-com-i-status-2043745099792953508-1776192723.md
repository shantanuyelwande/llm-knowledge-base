# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) without [[JavaScript]] enabled in the web browser. The error message indicates that JavaScript is a mandatory requirement for using the X.com platform and provides guidance for resolution.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents further browsing and interaction with the service.

## Error Details

### Error Message

The primary error message states:

> "We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Cause

The error occurs when:
- [[JavaScript]] is disabled in the user's browser settings
- The user is accessing X.com through an unsupported or outdated browser
- [[Browser extensions]] related to privacy may interfere with JavaScript functionality

### Secondary Issues

According to the error page, certain [[privacy-related browser extensions]] may cause compatibility issues with X.com, even when JavaScript is enabled. Users experiencing persistent problems are advised to disable such extensions temporarily.

## Resolution

### Recommended Actions

1. **Enable JavaScript** - Activate JavaScript in browser settings
2. **Switch Browsers** - Use a [[supported browser]] from X.com's approved list
3. **Disable Extensions** - Temporarily disable privacy-related extensions
4. **Clear Cache** - Attempt to reload the page using the "Try again" option

### Supported Browsers

For a complete list of supported browsers, users should consult the [Help Center](https://help.x.com/using-x/x-supported-browsers).

## Related Policies

## Technical Context

[[JavaScript]] is a client-side programming language essential for modern web applications. X.com's reliance on JavaScript enables dynamic content loading, real-time updates, and interactive features that define the platform's user experience.
