# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when attempting to access [[X (formerly Twitter)]] without [[JavaScript]] enabled in the user's browser. The error page provides guidance for users experiencing this issue and outlines the platform's browser support requirements.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error message preventing further use of the service. This is a common technical requirement for modern web applications that rely on client-side scripting for dynamic functionality.

## Error Details

### Error Message

The primary error notification states:

> "JavaScript is not available. We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com."

### Error Causes

The JavaScript requirement error may occur due to:

- **Disabled JavaScript**: Users have explicitly disabled JavaScript in their browser settings
- **Browser Incompatibility**: The user is accessing X.com through an unsupported or outdated browser
- **Browser Extensions**: [[Privacy-related browser extensions]] may interfere with X.com's functionality and trigger this error

## Solutions

### Recommended Actions

1. **Enable JavaScript**: Users should enable JavaScript in their browser settings
2. **Switch Browsers**: Use a [[web browser]] listed in X.com's supported browsers list
3. **Disable Extensions**: Remove or temporarily disable privacy-related browser extensions that may cause conflicts

### Resources

- **Help Center**: Information on [[X supported browsers]] and technical requirements is available in the X.com Help Center
- **Browser Support List**: A complete list of supported browsers can be found in the official documentation
