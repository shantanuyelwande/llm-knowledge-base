# X (Formerly Twitter) JavaScript Requirement

## Summary

This article documents a technical error message encountered when accessing X (formerly Twitter) with JavaScript disabled in a web browser. The platform requires JavaScript to function properly and will display an error page directing users to enable it or switch to a supported browser.

## Overview

X.com is a [[social media platform]] that requires [[JavaScript]] to be enabled in order to function. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents further browsing and provides guidance on how to resolve the issue.

## Technical Requirements

### JavaScript Dependency

X.com requires [[JavaScript]] to be active in the user's web browser. This is a common requirement for modern web applications that rely on client-side scripting to render content dynamically and handle user interactions.

### Browser Compatibility

The platform only functions on supported browsers. Users experiencing issues are directed to consult the Help Center for a list of compatible browsers. Common supported browsers typically include:

- Chrome
- Firefox
- Safari
- Edge

## Error Resolution

### Enabling JavaScript

Users encountering the "JavaScript is not available" message can resolve the issue by:

1. Enabling JavaScript in their browser settings
2. Checking browser extensions that may block JavaScript
3. Disabling privacy-related extensions that may interfere with the platform
4. Switching to a supported browser if their current browser is incompatible

### Privacy Extensions

Some [[privacy-related browser extensions]] may cause functionality issues on X.com. The platform recommends disabling these extensions before attempting to access the site again.
