# X.com JavaScript Requirement Error

## Summary

This article documents a technical error encountered when accessing X.com (formerly Twitter) without [[JavaScript]] enabled in the browser. The error page provides guidance on enabling JavaScript and lists supported browsers for proper platform functionality.

## Overview

X.com requires [[JavaScript]] to function properly. When a user attempts to access the platform with JavaScript disabled, they encounter an error page that prevents them from using the service.

## Error Details

### Error Message

The primary error message states:

> We've detected that JavaScript is disabled in this browser. Please enable JavaScript or switch to a supported browser to continue using x.com.

### Common Causes

- **JavaScript Disabled**: The user has manually disabled JavaScript in their browser settings
- **Browser Incompatibility**: The browser being used is not on X.com's list of supported browsers
- **Browser Extensions**: Certain [[privacy-related browser extensions]] may interfere with JavaScript functionality on the platform

### Troubleshooting Steps

Users encountering this error should:

1. Enable JavaScript in their browser settings
2. Verify their browser is [[supported by X.com]]
3. Disable privacy-related extensions that may cause conflicts
4. Attempt to access the platform again

## Supported Browsers

For a complete list of supported browsers and technical requirements, users are directed to the [X.com Help Center](https://help.x.com/using-x/x-supported-browsers).

## Technical Context

[[JavaScript]] is a fundamental technology required for modern web applications. X.com's reliance on JavaScript enables:

- Dynamic user interface rendering
- Real-time content updates
- Interactive features (posting, liking, retweeting)
- User authentication and session management
